看到这个效果的时候有点顿悟的感觉，因为平时写border的时候都是一条border:1px solid #000带过，整个border的效果是一起设置的，脑海里对于border的想象就是一个框，如下：

虽然也知道border分为border-top/border-right/border-bottom/border-left，但是从来没有试过考虑这四条边是怎么分的，潜意识里可能会觉得是四条边都是矩形，然后叠在一起了

也就是说一开始我想的边框是左边这样的，实际上是右边这样的，其实生活中也可以发现边框应该是右边这样的，比如常见的木质相框。技术源于生活啊。

当content的长宽为0时，边框就从梯形变成三角形，然后我们要做的就是隐藏其它的三条边。

如果要构造一个向下的三角形，那么最终展示的就是border-top那条边，与其相关的还有border-left和border-right，border-bottom就不需要了，将其宽度设置为0，效果如下

如果黄色和黑色不显示那么三角形就出来了，于是将border-left和border-right设置成透明的。实现了第一张图所示的最终效果。
