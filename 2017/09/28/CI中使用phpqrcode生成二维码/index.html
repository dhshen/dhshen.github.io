前言

在网页上展示二维码，前后端都有其实现的方式与库，那么在PHP CI框架中怎么去生成二维码呢，最终选择了比较流行的库phpqrcode来实现。

phpqrcode

官网：http://phpqrcode.sourceforge.net/

官网提供了演示DEMO和下载

PHP环境必须开启支持GD2

在CI中使用phpqrcode库

从官网下载包后解压，只需要其中的phpqrcode.php文件就好了，将其放入CI框架的/application/libraries目录下。

phpqrcode.php提供了一个关键的png()方法：

其中参数$text表示生成二维码的的信息文本；

参数$outfile表示是否输出二维码图片 文件，默认否；

参数$level表示容错率，也就是有被覆盖的区域还能识别，分别是 L（QR_ECLEVEL_L，7%），M（QR_ECLEVEL_M，15%），Q（QR_ECLEVEL_Q，25%），H（QR_ECLEVEL_H，30%）；

参数$size表示生成图片大小，默认是3；

参数$margin表示二维码周围边框空白区域间距值；

参数$saveandprint表示是否保存二维码并 显示
