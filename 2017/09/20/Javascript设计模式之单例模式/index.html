单例模式的定义是：

保证一个类仅有一个实例，并提供一个访问它的全局访问点

它的核心是确保只有一个实例，并提供全局访问

JavaScript中的单例模式

前面提到的更接近传统面向对象语言中的实现，但JavaScript是一门无类语言，生搬单例模式的概念并无意义。

全局变量不是单例模式，但在JavaScript开发中，经常会把全局变量当成单例模式

惰性单例

惰性单例指的是在需要的时候才创建对象实例。

惰性单例是单例模式的重点。
