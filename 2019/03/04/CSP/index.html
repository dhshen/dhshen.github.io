内容安全策略CSP

By dhshen

发表于 2019-03-04

简介

CSP（Content Security Policy）内容安全策略。

CSP的实质是白名单机制，明确告诉浏览器，哪些外部资源可以加载和执行。

CSP的主要目标是减少和报告XSS攻击。（注意：并不是防止或避免）

CSP配置实施

CSP可以在服务器端以添加HTTP头的方式或在网页的meta标签中设置的方式来实施。

CSP的配置有两条，分别是：

Content-Security-Policy：不符合CSP的外部资源会被阻止加载。

Content-Security-Policy-Report-Only：不执行限制选项，只是记录违反限制的行为。它必须与report-uri选项配合使用，其他规则与Content-Security-Policy相同。
